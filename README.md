# Migrance

Migrance is a web application designed to help users plan migration or travel by providing structured guidance, country insights, and relevant resources in one place. The platform also includes an AI-powered chatbot to assist users with their queries and real migration experiences to help with decision-making.

Built with React (Vite), the application focuses on clarity, usability, and a smooth user experience with a clean dark interface.

---

## Features

- AI-assisted chatbot for user queries  
- Step-by-step migration and travel guidance  
- Country comparison (cost, jobs, visa, etc.)  
- Interactive world map  
- Real migration stories  
- Centralized resource access  
- Basic analytics and insights  

---

## Tech Stack

- React (Vite)  
- React Router  
- Framer Motion  
- Recharts  
- React Simple Maps  
- CSS (Dark Theme)

## Installation

git clone https://github.com/balamurali462007/migrance.git  
cd migrance  
npm install  
npm run dev

## Usage

Start the development server
